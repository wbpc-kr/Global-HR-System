# WBPC-KR – Genesis AI 기반 보안 아키텍처

WBPC-KR는 외국인 근로자의 글로벌 현장 적용과 한국 생활을 지원하는 AI 기반 통합 플랫폼을 개발·운영하는 기관입니다.  
본 저장소는 WBPC 독자 AI **제니시스(Genesis)**를 기반으로 한 **위성 위장형 고스트 서버 보안 아키텍처**를 기록·운영합니다.

---

## 🔐 핵심 구성
- 제니시스 AI: 침입 탐지, 자동 미로 서버 생성, 특허 지원
- 고스트 서버: 메인 서버 위장, 외부 노출 없음
- 미로 서버: 침입자 무한 갇힘 구조, 자동 증식
- 보조 서버: 사용자 접속 지점, GitHub Pages 기반 운영
- 자동화: GitHub Actions + Kubernetes 연동 가능
