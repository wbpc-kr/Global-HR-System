# Global-HR-System: Part 1 Master Logic

## 1. Smart Gate (S0)
- 기기 인식 기반 신규/재방문 자동 분기 로직
- 1인칭 문 열림 및 실시간 기상(-1°C, 대설) 동기화

## 2. Dynamic Character (S1)
- 중앙 등장 연출 및 기후 맞춤 복장(3~4종) 자동 전환

## 3. Hourly Notification (S2)
- 매시간 정각 5분간 점멸 스타일 시간/날씨 정보 노출

## 4. VIP Escort & Arrival (S3, S4)
- 게스트 전용 엘리베이터 의전 및 VIP 미팅룸 안내
- 이전 상담 기록 기반의 연속성 강조형 AI 대화
