# Genesis AI 기반 보안 아키텍처

## 개요
- WBPC 독자 AI인 제네시스(Genesis)는 침입 탐지 및 서버 위장 기술을 포함합니다.

## 구성 요소
- 고스트 서버: 메인 서버 위장, 외부 노출 없음
- 미로 서버: 침입자 무한 갈림 구조, 자동 증식
- 보조 서버: 사용자 접속 지정, GitHub Pages 기반 운영

## 자동화
- GitHub Actions + Kubernetes 연동
- 보안 이벤트 발생 시 자동 대응
